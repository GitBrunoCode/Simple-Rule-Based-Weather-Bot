## Introduction

The goal of this project was to create a simple ChatBot capable of answering simple questions about the weather. A rule based ChatBot is used to answer correctly to the user thanks to regex. The bot is host on Node.js

## Installation




&nbsp;&nbsp;

### ***Create the API Key***
<img src=https://weatherstack.com/site_images/weatherstack_logo_footer.png alt="alt text" width="300" height="whatever" align="right">

* Go to [weatherstack](https://weatherstack.com/)
* Create a Free acount
* Go to *Control Panel - Dashboard*
* Save Your API Access Key for later


&nbsp;&nbsp;




### ***Launching the ChatBot***
<img src=https://upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Node.js_logo.svg/590px-Node.js_logo.svg.png alt="alt text" width="150" height="whatever" align="right">

* Download the main branch files
* Paste them in a new folder named ***project_name***
* Open the weather/index.js file in a code editor (example: Visual Studio Code)
* Edit the file with your custom key
&nbsp;&nbsp;

<a href="url"><img src="https://github.com/GitBrunoCode/Simple-Rule-Based-Weather-Bot/blob/main/app_key.png" align="center" width=80% height=80% ></a>




* Download and Install Node.Js if you haven't done it yet 
>[click here to download Node.Js ](https://duckduckgo.com)
* Use the following command in the Windows command prompt
```
   cd "project_name"
```
* Check that the current working directory is set to ***project_name*** folder 
* Use the following command in the Windows command prompt
```
   node app.js
```
>write the previous command again if you want to start a new conversation
